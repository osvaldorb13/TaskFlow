# TaskFlow

Plataforma de gestion de proyectos y tareas construida como proyecto de aprendizaje de desarrollo web full stack.

## Descripcion

TaskFlow es una aplicacion web completa que permite:

- Gestionar proyectos con tableros Kanban
- Crear y asignar tareas con prioridades
- Colaborar en equipo con roles y permisos
- Visualizar progreso con dashboards y reportes
- Modo claro y oscuro

## Stack Tecnologico

### Frontend

- **HTML5** — Estructura semantica y accesible
- **CSS3** — Variables CSS, Flexbox, Grid (sin frameworks)
- **JavaScript** — ES6+, vanilla

### Herramientas

- **Git + GitHub** — Control de versiones
- **W3C Validator** — Validacion HTML/CSS
